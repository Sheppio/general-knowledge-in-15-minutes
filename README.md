# General Knowledge in 15 Minutes

A library of fact-checked, audio-ready scripts on important subjects — each approximately 15 minutes when read aloud (~2,000–2,500 words).

## What This Is

Each piece is written in flowing prose with no bullet points, headers mid-text, or formatting that would sound awkward when read aloud. Every script is fact-checked against reliable sources (Wikipedia, Britannica) before being added to the library.
